# Color Switch Replica

### Replica of the Color Switch game</br>


### CONTROLS
Use Mouse left button or space button to jump.</br>
Use Escape key or pause button to go to pause menu.</br>

### TOOLS USED
* Unity 2019.1</br>
* Adobe photoshop CC</br>
